咚雞點餐系統－店長可操作交付版 v1.0.2

啟動需求：Node.js 22.17 或以上
安裝：npm install
開發：npm run dev
正式建置：npm run build
正式啟動：npm run start
完整驗證：npm run verify
瀏覽器流程驗證：npm run test:e2e

交付內容：
1. Next.js／React／TypeScript 完整原始碼
2. 顧客菜單、商品客製、購物車、結帳、訂單追蹤
3. 一起點建立、房間、參與者餐點與個人金額
4. 店長可新增完全空白分店、獨立分區、菜品與圖片
5. 分店發布、暫停、恢復與顧客端同步顯示
6. 跨店購物車與房間清理、歷史訂單保留
7. 領域狀態機、持久化、圖片與端對端測試
8. Vercel 設定、店長操作手冊、規格與驗證報告
9. 手機與桌面實際瀏覽器截圖

資料範圍：
本版以瀏覽器 localStorage 提供完整單機流程。正式多裝置營運需接入伺服器資料庫、驗證、即時服務、物件儲存與正式 LINE Pay；正式服務憑證不包含在交付包內。
